Quick Start

Up and running in minutes

Get started with just a few lines of configuration.

1

Add the Theme

Add remote_theme: devuri/ceres-theme to your _config.yml file.

2

Configure

Set up your site title, navigation, footer links, and social icons.

3

Build

Create pages using the provided layouts and components. Push to GitHub.
