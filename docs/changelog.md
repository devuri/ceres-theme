## [1.0.0] - 2025-01-01

### Added

- Initial release of Ceres Theme
- **Layouts**
  - `default` - Base layout with header and footer
  - `home` - Full-width landing page layout
  - `page` - Standard content page layout
- **Components**
  - Hero section with gradient text and stats
  - Bento grid for feature showcases
  - Steps grid for processes
  - Use cases grid for benefits/audiences
  - Pricing cards with feature lists
  - CTA section for conversions
- **Includes**
  - `header.html` - Navigation header
  - `footer.html` - Site footer with columns
  - `hero.html` - Hero section include
  - `bento-card.html` - Bento card include
  - `pricing-card.html` - Pricing card include
  - `cta.html` - CTA section include
- **Styling**
  - CSS custom properties for easy theming
  - Space Grotesk typography
  - JetBrains Mono for code
  - Indigo accent color palette
  - Full gray scale
  - Responsive breakpoints
  - Utility classes for spacing and alignment
- **Features**
  - GitHub Pages remote theme support
  - Configurable navigation
  - Configurable footer columns
  - Social media links
  - Custom CSS support
  - SEO tag support
- **Documentation**
  - Getting started guide
  - Configuration reference
  - Layout documentation
  - Component documentation
  - CSS reference
  - Color reference
  - Customization guide
  - Migration guide

## Reporting Issues

Found a bug or have a feature request?

[Open an issue](https://github.com/devuri/ceres-theme/issues/new) on GitHub with:

- Clear description of the issue
- Steps to reproduce (for bugs)
- Expected vs actual behavior
- Screenshots if applicable
- Your Jekyll version and environment
